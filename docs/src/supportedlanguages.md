# Supported languages

At time of writing, seedmixer supports the following languages:

- English (eng)
- Spanish (spa)
- French (fra)
- Italian (ita)
- Chinese Simplified (zho-hans)
- Chinese Traditional (zho-hant)
- Czech (cze)
- Japanese (jpn)
- Korean (kor)
- Portuguese (por)

If you want seedmixer to support another language - please raise an issue.
